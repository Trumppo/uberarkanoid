# uberarkanoid

Tämä repositorio on perustettu uuden Arkanoid-tyyppisen pelin kehitystä varten.
Seuraavat askeleet ovat vielä työn alla:

- ideoidaan pelin mekaniikka ja tasosuunnittelu
- valitaan joko JavaScript/Canvas, Unity, Godot tai muu pelimoottori
- rakennetaan ensimmäinen prototyyppi ja testataan peruskuulat, padit ja seinät
- kytketään pelimekaniikka takaisin OpenClaw-työkaluun

Tällä hetkellä repo toimii tyhjänä pohjana tulevalle pelikehitykselle.
